Getting Started

Welcome to the Fuzzy Learning Bot admin panel. Here's how to use it:

1. Ask a Question

Type a question in the "Ask a Question" section to get an answer from the knowledge base.

2. Teach New Knowledge

Add new question-answer pairs to teach the bot. Provide multiple answers separated by commas.

3. Bulk Upload

Upload multiple questions and answers at once using JSON format:

[
  {
    "question": "What is AI?",
    "answers": ["Artificial Intelligence"]
  },
  {
    "question": "What is ML?",
    "answers": ["Machine Learning"]
  }
]

4. Delete Knowledge

Remove incorrect or outdated information from the knowledge base.

5. Analytics

View statistics about your knowledge base and teacher contributions.
